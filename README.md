# Evolutionary-Computing-BoxPacking-Problem-Java
<h1>Box packing Problem - Machine learning</h1>

<h2>Description</h2>
Project consists of a Java program to solve the problem of sorting boxes of varying dimensions into the back of a lorry with varying dimensions using machine learning. The goal is to put all boxes into the lorry so that all boxes fit neatly with no gaps between them whilst "sliding" from the left to the right.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Java</b> 

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
